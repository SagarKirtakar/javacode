# Java OOPs and Logical Programs

This repository contains Java programs covering Object-Oriented Programming (OOP) concepts and commonly asked logical coding problems.

## 📚 Topics Covered

### Object-Oriented Programming (OOP)

* Classes and Objects
* Constructors
* Inheritance
* Polymorphism
* Abstraction
* Encapsulation
* Interfaces
* Abstract Classes
* Method Overriding and Overloading

### Logical Programs

* Arrays
* Strings
* Numbers
* Pattern Programs
* Searching and Sorting
* Duplicate Elements
* First Non-Repeated Character
* Second Largest Element
* Palindrome Programs
* Prime Numbers
* Fibonacci Series
* Factorial and More
